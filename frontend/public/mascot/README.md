Mascot asset contract
=====================

Place transparent PNG files in this folder with these exact names:

- mascot-idle.png
- mascot-listening.png
- mascot-talking.png
- mascot-thinking.png

Recommended export settings:

- Real transparent alpha background (no checkerboard in the image itself)
- Square canvas: 512x512 or 1024x1024
- Consistent mascot scale and center alignment across all files
- 8-12 percent internal padding around the character

Notes:

- If one or more files are missing, the app falls back to `/images/servo_base_image.png`.
- You can replace assets anytime without code changes as long as names stay the same.
